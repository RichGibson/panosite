Command Line Tools

  • load_gigapans.py
  • get_json.py Get the .json file for every gigapan in json/
  • load_json.py Load Gigapan model from files in json/
  • check_json.py Shows summary of gigapans in json/
  • aspect_ratio.py extracts gigapan_id, width, height in order to run stats.
  • crop_work.py work on automatically cropping a gigapan
  • tile_foo.py Methods to work with tiles, getTileName and URL from (id, level, x, y). to be integrated into a library of tile utilities.
  • index_table.py Makes a table showing tile names for each tile in the quadtree model. Just for understanding the quadtree structure
